# data-engg-assignment
These project extracts data from SQLite database and calculates total quantities of items purchased by customer with age 18 to 35

Two solutions:
1.sql-based python script
2.pandas-based python script

Total files:
1.sql.solution.py
2.pandas solution.py
3.Data Engineer_ETL Assignment.db 

In Output:
Both scripts generate a csv file with format as
customer;age;item;quantity

requirement.txt:
Install dependencies:
pip install -r requirements.txt

then run:
python sql_solution.py
python pandas_solution.py
